

# Personal-Blog-Site



To run it locally, first run `npm install` in terminal to install the dependencies from package.json file



To run the server command is `node app.js` 



For testing, it is recommended to run the server using `nodemon app.js`




About Usage:



This is a multipage personal blog website, go to '/compose' directory to publish a new blog.



![image](https://user-images.githubusercontent.com/98037929/229453564-80704d30-0faf-48bf-b32e-a4f9b7b86e88.png)





Templates and Layouts are made using <a href="https://ejs.co/"> EJS </a>



The Backend server is built using <a href="https://nodejs.org">Node.js</a> and <a href="https://expressjs.com/">Express.js</a>.
